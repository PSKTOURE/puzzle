Les commandes principales sont :

- "sh scripts/run.sh [nbElements]" pour lancer le programme. Mettre aucun n'argument pour lancer la version en ligne de commande
				    Mettre n'importe quel argument (exemple : 1) pour lancer la version avec GUI.
				    La version avec GUI possède un menu permettant de changer d'image.
- "sh scripts/makejar.sh" pour créer un jar contenant tous les .class
- "sh scripts/makedoc.sh" pour générer la javadoc (le fichier principal est alors doc/index.html)
- "sh scripts/clean.sh" pour supprimer tous les fichiers générés
